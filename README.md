XcelerateStats is a data-driven football analytics project that analyzes player and team performance using Exploratory Data Analysis (EDA) and Machine Learning. The project aims to extract meaningful insights from football datasets and support performance evaluation, match outcome analysis, and training recommendations.

* Project Objectives

Analyze football player and team performance using real match data
Identify key performance indicators (KPIs) influencing wins and losses
Apply machine learning models for performance and training analysis
Provide data-backed insights to support coaching and strategy decisions

* Exploratory Data Analysis (EDA)

Examined dataset structure, data types, and missing values
Analyzed player statistics, team performance, and match schedules
Visualized key metrics affecting performance and match outcomes

* Machine Learning & Analysis

Built models to analyze training and performance trends
Evaluated player efficiency and team strengths
Identified factors influencing match wins and losses

Clear insights into player and team performance
